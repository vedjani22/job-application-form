# Job Application Form

This project is a simple and responsive job application form designed using HTML5 and CSS3. It simulates a real-world application form where users can enter their details and apply for a position.

# Overview

The form includes multiple input types and is structured to provide a smooth and user-friendly experience. It focuses on layout design, form structure, and clean styling.

# Key Features

* User input fields (name, email, phone)
* Job role selection (dropdown menu)
* Skills selection using checkboxes
* Resume upload interface
* Cover letter section
* Clean and responsive UI design

* Built With
* HTML5
* CSS3
